Power plant function description

The main component of hybrid power plants is the Hybrid inverter, which has input from batteries for island mode and input from the distribution network for switching to this backup.

It is a voltage inverter and charger in one. It includes a pure sine inverter, adaptive charging, PowerAssist hybrid technology, and other features for integration into various types of systems. It is equipped with one AC input for connecting the distribution network or e.g. a generator. The AC1 output ensures power supply without interruption, e.g. in UPS mode, MultiPlus takes over the power supply of connected appliances in less than 20 milliseconds in the event of a distribution network or generator failure. The second output is functional only if the AC input is connected to the distribution network or generator. A load that would quickly discharge the batteries can therefore be connected to this output and will only be powered from the AC source.

Hybrid photovoltaic power plant, electric vehicle chargers, rainwater storage system for utility water

  • Hybrid photovoltaic power plants with electric vehicle chargers. Those parking in your parking lot can simultaneously charge their electric vehicle.
  • Energy is distributed directly from the power plant to the charger, supplemented by consumption from the battery storage if necessary, and if the supply from the power plant is insufficient, the system can supplement the consumption from the external distribution network.
  • To prevent the grid consumption from being too sudden and to avoid large fluctuations in consumption, the system can be set so that the batteries will be partially charged from the grid during periods when the power plant is not producing.
  • The entire system can be controlled, and if more consumers connect for fast charging of electric vehicles, the system will distribute capacity for individual chargers.
  • In addition to the systems already described, it is possible to add a tank for collecting rainwater, which can be used as utility water. The overflow from the tank is connected to the rainwater drainage.
